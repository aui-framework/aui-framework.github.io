Devtools

Devtools is a UI set of tools integrated in aui.views and thus in every AUI application. The devtools window can be invoked by hitting CTRL+F12 on AUI-based application.

Layout tab

The most notable functionality of devtools is the Layout tab. Here you can discover your view hierarchy and check which ASS rules were applied to your view.

Disabling devtools in production

By default, devtools window is accessible in both debug and release builds of AUI applications so in theory it can be accessed by users of your application. If it renders unacceptable for you, you can override and stub AWindowBase::createDevtoolsWindow method in your AWindow.
