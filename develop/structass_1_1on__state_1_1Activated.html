Detailed Description#

It's convenient to use it with with_style:

_new<AView>() with_style {
  MinSize { 16_dp },
  BackgroundSolid { AColor::BLACK },
  on_state::Activated {
    BackgroundSolid { AColor::RED },
  },
},
