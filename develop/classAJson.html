AJson AJson::mergedWith ( const AJson & other)

Merges other json object into this object.

Let's say this json is:

{
     "key0": "val1",
     "key1": "val1",
}

And other json is:

{
     "key1": "val2",
     "key2": "val2",
}

Then the result is:

{
     "key0": "val1",
     "key1": "val2",
     "key2": "val2"
}
