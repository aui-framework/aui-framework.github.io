Detailed Description

CSS-like styling system.

AUI uses a CSS-like stylesheet domain specific language, ASS (stands for Aui Style Sheets). Like CSS, ASS is a list of rules. A rule consists of a selector and a list of declarations. Selector is a basic matcher that determines whether apply a rule to the specific view or not. Declaration controls the specific aspect of view's style (i.e. FontSize { 18_pt } equals "call setFontSize(18_pt) for every matched view").

CSS AUI ASS Meaning
AButton {
background: red;
}
{
t<AButton>(),
BackgroundColor { AColor::RED },
},
Set all buttons' background to red
ATextField::hover {
border: 1px solid #00f;
}
{
t<ATextField>::hover(),
Border { 1_dp, 0x0000ff_rgb },
},
Set all text fields' border to #00f (blue) on hover
